### 软件功能
此软件是网络中的二手交易平台，每个人都有发布自己二手商品的权限，以及管理自己的商品，对自己的商品进行修改和删除操作。<br>
有了发布自己的商品，那么就必须有购买别人的商品功能，在本系统中任何人都能看见所有人发布的所有商品，从而在其中挑选自己所需的物品，点击物品会弹出该商品的主人的联系方式，从而主人联系进行二手交易。
